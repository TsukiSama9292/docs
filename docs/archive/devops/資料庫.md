## DBeaver

### 安裝 DBeaver

```bash
sudo mkdir -p /etc/apt/trusted.gpg.d

# 下方修正為單反斜線，或直接連成一行
curl -fsSL https://dbeaver.io/debs/dbeaver.gpg.key | \
  sudo gpg --dearmor -o /etc/apt/trusted.gpg.d/dbeaver.gpg

echo "deb [signed-by=/etc/apt/trusted.gpg.d/dbeaver.gpg] https://dbeaver.io/debs/dbeaver-ce /" | \
  sudo tee /etc/apt/sources.list.d/dbeaver.list

sudo apt update
sudo apt install dbeaver-ce -y
```

## **Redisinsight - Redis/Valkey 連接工具(Redis 官方製作)**

## docker-compose.yml

```yaml
x-default-opts: &default-opts
  restart: unless-stopped
  tty: true
  stdin_open: true
  privileged: false
  ipc: private

services:
  redisinsight:
    <<: *default-opts
    image: redislabs/redisinsight:latest
    container_name: redisinsight
    ports:
      - "5540:5540"
    volumes:
      - redisinsight:/data

volumes:
  redisinsight: {}
```

## 快速啟動

```bash
docker compose up -d
```

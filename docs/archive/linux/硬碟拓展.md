# 硬碟拓展

建立時間: 2026年2月3日 下午7:51
類別: Linux
最後更新時間: 2026年2月3日 下午7:51

```
# 安裝工具
sudo apt update
sudo apt install -y cloud-guest-utils

# 更新 PV - 這裡是 sda3 要拓展
sudo growpart /dev/sda 3
sudo pvresize /dev/sda3

# 確認
sudo pvs
sudo vgdisplay

# 同步拓展
sudo lvextend -r -l +100%FREE /dev/ubuntu-vg/ubuntu-lv

# 最後檢查
lsblk
df -hT
vgdisplay
```

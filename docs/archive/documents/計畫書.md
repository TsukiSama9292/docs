## 主題：基於OOOO和OOOO研究是否會對影響OO效果（我瞎掰的，但應該有這樣的格式）

## (二) 研究動機與研究問題

OOOO(核心問題)很重要(好像要毀天滅地一樣) ->
現在OOOO方式有OOOO的困境(好像很難突破一樣) ->
當前OOOO技術有OOOO的優勢(不會到很稀罕，但是要確切) ->
根據現有的研究結果，OOOO技術能減輕此問題的困境或其他問題的困境(這裡要找論文或是研究報告) ->
但是現在的OOOO方面仍有不足之處 ->
本研究根據此情形，提出以OOOO作為核心思想，基於OOOO和OOOO的實驗理論與結果，進行什麼樣的設計與改良，力求獲得什麼樣的結果。

## (三) 文獻回顧與探討

- 核心問題很重要
    - 報告、研究、新聞
    - 該問題不利於台灣...
- 次技術/方法(很可能不是程式碼，但對於核心問題有很深的關係)
    - 報告、研究
- 核心技術
    - 強調如何透過這技術搭配新的設計或改良解決*核心問題*

## (四) 研究方法與步驟

- 研究工具(舉例)
    - 問卷：使用前後各一次，調查：學習動機、學習表現...
    - 測驗：測量特定能力或知識，調查：學習表現
    - 訪談：隨機抽取幾位使用者做訪談，調查：獲取使用者對於產品直觀想法
    - 觀察表：研究者設計的觀察指標或評分表，紀錄特定行為、互動或事件
    - 分析工具(or 模型評估框架)：K-Means 之類的
    - 實驗器材：自然科學或工程研究的設備
    - 軟體、平台：可能是自己開發的
- 研究步驟(流水式的寫法，以下是舉例)
    1. 徵求實驗對象(60名)：大學生、修過某個課程的學生
    2. 對實驗對象進行問卷調查、測驗
    3. 將實驗對象隨機拆分成兩組(每組30名)
    4. 讓實驗對象使用自己的服務或產品，幾個禮拜
    5. 對實驗對象進行問卷調查、測驗、隨機抽取實驗對象進行訪談
    6. 進行數據統計，統整出實驗結果
